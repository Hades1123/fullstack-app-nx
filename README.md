**Cách chạy dự án**

# 1: `Clone` từ github về trước nhé

# 2: npm i

# 3: Cấu hình các file .env trong folder api và ui theo .env.example

# 4: npm run prisma:init

# 5: npx nx serve ui

**Note**

# 1: Ở bước 5 -> chạy cả BE và FE cùng lúc. Tất nhiên là chạy FE thì phải chạy luôn cả BE luôn chứ nhỉ :D

# 2: Chạy BE thôi --> npx nx serve api

# 3: Bên BE báo Daemon ko hoạt động, ko thể chạy lại khi có file bị chỉnh sửa ?

--> npx nx reset

# 4: nếu #3 ko sửa được thì chạy thêm lần nữa xem sao, ko thì search gemini hỏi thêm thử . Lỗi này cx ko quan trọng lắm. Tốn công bấm th :D.
